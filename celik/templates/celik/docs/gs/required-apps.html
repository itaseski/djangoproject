Required applications

Python 3.11 and VS Code 1.74

On this page
  • Install Python
  • Install VS Code
  • Install Django
  • Create the Django project
  • Create empty DB
  • Verify-project
  • Open browser

Install Python

Being a Python web framework, Django requires Python.

Get the latest version of Python at https://www.python.org/

Install VS Code

Free and built on open source. Integrated Git, debugging and extensions.

Download Visual Studio Code at https://code.visualstudio.com/download

Install Django 

$ python -m pip install --upgrade pip
$ python -m pip instal django

Create the Django project 

$ python django-admin startproject web_project .

Create an empty development database

Next, run the Django command-line utilities to create the database tables automatically:

$ python manage.py migrate

Verify the Django project 

$ python manage.py runnserver

Open browser

Ctrl+click the http://127.0.0.1:8000/

Stop the server

in VS Code using Ctrl+C
